# FabioApp
Um projeto baseado em Qt com objetivo de aprimorar os conhecimentos em Python.

## Execução / Instalação
Execute o código fonte ou baixe o executável extraindo em outro local.
<br><br>
<b>Requisitos:</b>
- Python 3.12
- Windows (testado)
- PySide6

<b>Instalação da Biblioteca</b><br>
`pip install pyside6`
